# Implement — 기능 구현

## 목적

새 기능을 구현할 때 탐색 → 설계 → 구현 → 검증 순서로 진행합니다.
코드를 바로 작성하기 전에 반드시 기존 패턴을 파악합니다.

## 실행 절차

1. **컨텍스트 파악**
   - CLAUDE.md 아키텍처 규칙 확인
   - 관련 기존 코드 탐색
   - `.claude/decisions/`에서 관련 ADR 확인

2. **구현 범위 확인**
   - `.claude/tasks/`에 작업 상태 파일 있으면 해당 태스크 확인
   - 없으면 사용자와 범위 합의

3. **구현**
   - 기존 코드 패턴 준수
   - CLAUDE.md의 아키텍처 규칙 준수
   - 작은 단위로 구현 후 확인 반복

4. **검증**
   - 빌드 통과 확인
   - 린트 오류 없음 확인
   - CLAUDE.md의 체크리스트 항목 확인

5. **태스크 상태 업데이트**
   - `.claude/tasks/`에 진행 상태 기록
