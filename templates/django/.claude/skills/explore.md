# Explore — 코드베이스 탐색

## 목적

작업 시작 전 코드베이스의 구조, 패턴, 의존성을 빠르게 파악합니다.
CLAUDE.md를 먼저 읽고, 실제 디렉토리 구조를 탐색한 뒤 핵심 파일을 확인합니다.

## 실행 절차

1. **CLAUDE.md 읽기** — 프로젝트 개요, 아키텍처 규칙 확인
2. **디렉토리 구조 파악** — 최상위 2~3단계 구조 확인
3. **핵심 파일 식별** — 진입점, 설정 파일, 주요 모듈
4. **ADR 확인** — `.claude/decisions/`에서 과거 결정 참고
5. **요약 리포트** — 발견한 내용을 간결하게 정리

## 출력 형식

```
## 코드베이스 요약

**구조:** [디렉토리 트리 요약]
**진입점:** [주요 파일 경로]
**핵심 패턴:** [발견된 패턴]
**주의 사항:** [작업 전 알아야 할 것]
```
